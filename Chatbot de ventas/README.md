## Chatbot de ventas

--------- Preguntas para el desarrollador --------
### ¿Por qué se desarrolla este proyecto?

Se desarrolla con el objetivo de adquirir experiencia práctica en el desarrollo de chatbots con IA.

### ¿Qué problema resuelve?

Resuelve la automatización de procesos de ventas y registro de clientes.

--------- Preguntas para el cliente --------------
### ¿Cuál es el objetivo principal del Chatbot?

El objetivo principal es automatizar el proceso de ventas y registro de clientes.

### ¿En qué plataforma lo necesitan?

Se necesita para una pagina web que será diseñada por el mismo desarrollador.

### ¿Que tareas especificas debe realizar?

1) Registrar clientes con datos personales y dirección de envio.
2) Revisar la disponibilidad en stock del producto necesitado por el cliente.
3) Dar respuestas con respecto a la disponibilidad.
4) Cerrar el proceso de ventas en el caso de que hay disponilidad en stock y el cliente quiera avanzar en la compra.
5) Dar respuestas de cierre de ventas y dirección de envio.

### ¿Necesita integrarse con algún sistema?

Por definir. 

### ¿Qué resultado final se espera?

Ventas automatizadas y flujo conversacional para algún otro tipo de requerimiento relacionado a ventas.
